СВЕРКА EXCEL: ОФД / КАРТЫ / КАССА
==================================

Как запустить на Mac (без программирования)
-------------------------------------------

1. Распакуйте архив SverkaOFD-mac.zip (двойной щелчок).

2. Откройте папку и найдите файл «SverkaOFD.app».

3. При первом запуске Mac может показать предупреждение
   «разработчик не подтверждён»:
   • Откройте «Системные настройки» → «Конфиденциальность и безопасность»
   • Нажмите «Все равно открыть» для SverkaOFD.

   Или: правый щелчок по SverkaOFD.app → «Открыть» → «Открыть».

4. Откроется окно программы с четырьмя кнопками «Выбрать…».

Как пользоваться
----------------

1. Основной отчет ОФД (список чеков) — файл (1).xlsx
2. Доп. отчет ОФД по сменам — файл (2).xlsx (можно оставить пустым)
3. Реестр оплат платежной картой
4. Реестр кассовых документов

Нажмите «Сформировать сверку» и укажите, куда сохранить отчёт.

Результат — Excel с вкладками:
  • Сверка по дням
  • Итоги
  • Расхождения
  • Правила

Поддержка
---------
Если приложение не открывается — напишите отправителю архива.
