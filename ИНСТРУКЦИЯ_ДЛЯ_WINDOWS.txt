СВЕРКА EXCEL: ОФД / КАРТЫ / КАССА
==================================
Инструкция для Windows (без Python и без программирования)


КАК ЗАПУСТИТЬ
-------------
1. Распакуйте архив в любую папку (например, на Рабочий стол).

2. Дважды щёлкните по файлу:
      SverkaOFD.exe

3. Если Windows показывает предупреждение «Windows защитил ваш компьютер»:
   • Нажмите «Подробнее»
   • Затем «Выполнить в любом случае»
   (программа не подписана сертификатом — это нормально для внутренних инструментов)

4. Откроется окно «Сверка ОФД с реестрами».


КАК СДЕЛАТЬ СВЕРКУ
------------------
В окне программы нажмите «Выбрать…» и укажите файлы:

  1. Основной отчёт ОФД (список чеков)          — ОБЯЗАТЕЛЬНО
  2. Доп. отчёт ОФД по сменам                    — необязательно
  3. Реестр оплат платежной картой               — ОБЯЗАТЕЛЬНО
  4. Реестр кассовых документов                  — ОБЯЗАТЕЛЬНО

Затем:
  • Нажмите «Сформировать сверку»
  • Укажите, куда сохранить файл (например, сверка.xlsx на Рабочий стол)
  • Дождитесь сообщения «Сверка завершена»


ЧТО БУДЕТ В РЕЗУЛЬТАТЕ
----------------------
Excel-файл с вкладками:
  • Сверка по дням   — сверка по каждому дню
  • Итоги            — суммы по подразделениям
  • Расхождения      — только строки с отличиями (красным)
  • Правила          — описание правил сверки


ЕСЛИ НЕ ЗАПУСКАЕТСЯ
-------------------
• Убедитесь, что файл SverkaOFD.exe не заблокирован:
  правый щелчок → Свойства → внизу «Разблокировать» → ОК

• Напишите тому, кто передал вам программу.
